# FunctionFox report

This app generates sanitized, formatted json and csv files from an exported FunctionFox report

This is an open source project which uses the [The Hippocratic License][license].

## Quick Start

### Requirements

1. Node
  - Check if Node is installed: `node --version`
  - If you see a version number, such as `v11.6.0`, proceed to [Get Started](#get-started)
  - If Node isn't installed, [download][node-download] and install it, then proceed to [Get Started](#get-started)

### Get Started

Fork or clone this repo, install dev dependencies, and start:

```bash
git clone https://github.com/paulshryock/FunctionFox-Report.git
cd FunctionFox-Report
npm i
npm start
```
